# <img src="https://cdn.jsdelivr.net/npm/simple-icons@v9/icons/roblox.svg" width="24"> Kairo Avatar

Script desenvolvido para **Roblox** que permite equipar automaticamente **o avatar de outro jogador** no seu personagem.

---

## Funcionalidades

- Copia o avatar de outro jogador
- Equipa automaticamente no personagem
- Execução simples via script
- Script leve e rápido

---
